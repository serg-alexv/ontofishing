# ChatGPT adversarial audit prompt

Audit the candidate table using 0truth A+B.

A = constraints:
- official source exists
- status is actually open/rolling
- deadline is not passed
- independent researcher/engineer eligibility is plausible
- payout/funding exists
- deliverable is clear
- jurisdiction/payment constraints are acceptable

B = consumptions:
- time cost
- technical complexity
- probability of award/payout
- portfolio value
- compute/wet-lab/hardware cost
- admin burden
- psychological/attention cost

Return:
1. Execute this week
2. Prepare application
3. Monitor
4. Reject
5. Missing verification data
6. Best 14-day plan

# Grok Heavy discovery prompt

Run a 16-agent bounty-intelligence search for currently open, verifiable opportunities for an independent researcher/engineer in:

1. AI memory, agentic research automation, RAG, knowledge graphs, hallucination detection, AI evaluation.
2. AI safety, red-teaming, model behavior evaluation, agent security.
3. Privacy/network/security engineering, routing, censorship-resistance, protocol research, iOS NetworkExtension.
4. Open-source engineering: Rust, Python, TypeScript, C/C++, FreeBSD/Linux, developer tooling.
5. Biotech, LAB, genomics, metabolic engineering, synthetic biology, AI-for-science.

Reject closed, stale, no-payout, student-only, company-only, invite-only, institutional-nomination-only, unclear eligibility, unclear deliverable, and no official source.

Output a CSV-compatible table with:
name, official_url, domain, status, status_evidence, deadline, payout_amount, eligibility, deliverable, jurisdiction_or_payment_constraints, source_type, effort, domain_fit, verification_strength, payout_value, probability_of_success, portfolio_value, deadline_accessibility, low_admin_friction, eligibility_risk, payment_risk, stale_status_risk, competition_noise, notes.

Also produce a rejected-items table with reason.
